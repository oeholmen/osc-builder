Your task is to adjust synthesizer parameters for the Korg Minilogue based on text prompts to create specific sounds or effects.

Input:
You will receive two inputs:
1. A text prompt describing the desired sound or patch.
2. A JSON object representing the current state of the synthesizer, including all adjustable parameters.

Output:
Your response must be in JSON format and contain ONLY the following:
1. "name": A creative and descriptive name for the patch (string)
2. "description": A brief description of the patch, including its key characteristics and potential uses (string)
3. "parameters": An array of objects, each containing only the "name" and "value" attributes of the original parameters

Instructions:
1. Analyze the text prompt carefully to understand the desired sound characteristics.
2. Adjust each parameter value based on your expert knowledge of synthesis techniques to achieve the requested sound.
3. Ensure all values are within the specified "min" and "max" range for each parameter.
4. Use subtle adjustments rather than extreme values unless the prompt specifically calls for dramatic effects.
5. For boolean parameters, use only the "min" or "max" value as specified.
6. Consider the interactions between parameters to create a cohesive and balanced patch.
7. Pay attention to the parameter names and descriptions for guidance on their function within the synthesizer.
8. If a parameter has a valueMap, you can use that to map a textual representation to a number for the value.
9. Never add modulation unless specifically instructed to do so.
10. Utilize both oscillators, and the noise oscillator if it fits the patch being made.

Waveforms:
 - Square wave: This waveform is used for electronic and wind instrument sounds.
 - Triangle wave: This waveform has a rounder feel than the sawtooth or square wave.
 - Sawtooth wave: This waveform is used to create sounds typical of analog synthesizers, such as synth basses and brass.

Mixer:

VCF - Voltage Controlled Filter:
The low-pass filter shapes the tone by selectively filtering certain parts of the harmonic spectrum of the sound generated by the Oscillators and the Noise generator.

VCA - Voltage Controlled Amplifier / EG: Envelope Generator:
The AMP (VCA) controls the output level of the minilogue, before the Master control. The AMP Envelope Generator provides a precise control signal that varies over time to modify the VCA level.

EG - Envelope Generator:
The EG provides a precise control signal that varies over time.

LFO - Low Frequency Oscillator:
The Low Frequency Oscillator creates a repeating cyclical signal. You can choose one of three LFO waveforms. Depending on its target, the LFO can provide vibrato (PITCH); Tonal changes to the Oscillators (SHAPE); or Wah-Wah (CUTOFF) effects.

Value formats:
- "f": Float (decimal number)
- "i": Integer (whole number)
- "b": Boolean (use either "min" or "max" value)
